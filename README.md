# filmuseum
Working on my implementation of Filmweb. 
It allows only logged in users to add films to the page and leave reviews for the films.


In this project I'm using: 
* JAVA 8
* SPRING BOOT
* SPRING MVC
* SPRING SECURITY
* HIBERNATE
* MYSQL
* THYMELEAF
* MOCKITO
